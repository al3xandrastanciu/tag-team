# TagTeam
  Aplicație web de tip Single Page Application (SPA) dezvoltată pentru a facilita gestionarea și urmărirea defectelor (bug-urilor) într-un mod structurat, asigurând comunicarea eficientă între membrii echipei de dezvoltare și testeri.
  Tehnologii folosite: 
    - Frontend: React
    - Backend: Node.js cu Express
    - Baza de date: MongoDB
